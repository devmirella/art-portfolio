# 🎨 art-portfolio

Repositório de estudos e processos artísticos autorais.

Este espaço reúne desenhos produzidos ao longo do tempo — não como produto final, mas como registro de processo. Cada pasta representa uma linguagem ou foco de estudo diferente, refletindo uma prática constante de observação, experimentação e desenvolvimento técnico.

---

## Sobre

Sou Mirella, desenvolvedora front-end com forte senso estético formado por anos de prática em desenho e pintura. A arte não é um hobbie paralelo ao desenvolvimento — ela informa diretamente a forma como penso interface, composição visual e hierarquia.

Atualmente estudo front-end e UX, construindo projetos próprios onde código e estética se encontram. Esse repositório existe porque acredito que processo importa tanto quanto resultado.
